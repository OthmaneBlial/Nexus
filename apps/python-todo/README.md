# Python TODO API

FastAPI-powered TODO list showcasing a modular Python service. Includes routers, dependency injection, and tests.

## Getting started

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
uvicorn main:app --reload
```

## Run tests

```bash
pytest
```

Visit `http://127.0.0.1:8000/docs` for interactive OpenAPI docs.
